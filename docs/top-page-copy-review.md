# トップページ文言レビュー

レビュー日: 2026-06-14  
対象: `index.html`

## 総評

トップページ全体の方向性は、「耳をひらく」「音風景」「まちを歩く」という言葉の軸があり、イベントの世界観は伝わっています。特にヒーローコピーの「耳をひらくと、さいたまは別の風景になる。」は、活動の体験価値を短く言い切れており、残す価値があります。

一方で、告知ページとして見ると、やや詩的な表現が重なりすぎて、初めて訪れた人が「具体的に何をするイベントなのか」「自分でも参加できるのか」を判断する材料が少し不足しています。また、「氷川の森」「大宮公園」「大宮一番街」「大宮駅周辺」などのルート表記が複数あり、同じイベントを指しているのか迷う箇所があります。

優先して直すなら、以下の3点です。

1. ルート、集合場所、内容の表記を統一する
2. イベント説明を少し具体化し、初参加者向けの安心材料を足す
3. 過度に抽象的、重複的な言い回しを短くする

## 重要度高

### 1. ルート表記が揺れている

該当箇所:

- Hero: `大宮〜氷川の森`
- Event: `大宮の街から氷川の森へ`
- Event facts: `大宮駅周辺〜大宮公園`
- Project: `大宮一番街から氷川の森へ`
- Program: `大宮一番街〜氷川の森`

問題:

同じイベントのルートとして読めますが、表現が少しずつ違うため、参加者は実際にどこを歩くのか判断しづらくなります。特に「氷川の森」と「大宮公園」は、土地勘がない人には別地点にも見えます。

推奨:

表記を1つに統一し、必要に応じて補足を入れる。

修正文案:

> 大宮駅周辺から氷川参道、大宮公園周辺へ。街なかから森へ移る音の変化を聴きながら歩きます。

短縮版:

> 大宮駅周辺から氷川参道、大宮公園周辺へ。

### 2. 集合場所の確定度が揺れている

該当箇所:

- Event facts: `まちラボおおみや（JR大宮駅徒歩3分）`
- Program: `集合: まちラボおおみや（仮）`
- JSON-LD: `まちラボおおみや`

問題:

ページ内で「確定」と「仮」が混在しています。申し込み判断に直結する情報なので、どちらかに統一したほうがよいです。

推奨:

未確定なら全箇所で「予定」とし、確定後に外す。確定済みならProgramの「仮」を削除する。

修正文案:

> 集合: まちラボおおみや（JR大宮駅徒歩3分）

未確定の場合:

> 集合: まちラボおおみや予定（JR大宮駅徒歩3分）

### 3. 初参加者向けの説明が足りない

該当箇所:

- Event本文
- Join
- Program

問題:

イベントの魅力は伝わりますが、「録音機材が必要か」「経験がなくてもよいか」「どれくらい歩くか」「持ち物」など、参加前の不安を下げる情報が不足しています。Peatix側に詳細があるとしても、トップページにも最低限あると申し込み導線が強くなります。

追記候補:

> 録音経験や専用機材は不要です。スマートフォンだけでも参加できます。

> 当日は屋外を歩きます。飲み物、歩きやすい靴、必要に応じて録音できるスマートフォンやレコーダーをご用意ください。

> 音を聴くワーク、短い録音、参加者同士の共有を行います。

## 重要度中

### 4. ヒーロー説明文は少し情報を詰めすぎ

現文:

> フィールドレコーディングクラブさいたまは、まちの音、自然の音、人の気配を記録し、さいたまの“音”風景を未来へ残す市民クラブです。

評価:

方向性は良いですが、「まちの音、自然の音、人の気配」と「音風景」が近い意味で重なっています。また「未来へ残す」はやや大きな表現で、トップの短文としては少し抽象的です。

修正文案:

> フィールドレコーディングクラブさいたまは、まちや自然の音を記録し、さいたまの音風景を共有する市民クラブです。

詩的な余韻を残す案:

> フィールドレコーディングクラブさいたまは、まちや自然の音を記録し、ふだん聞き過ごしているさいたまの音風景を共有する市民クラブです。

### 5. Aboutの2文は内容が近い

現文:

> さいたま市を拠点に、フィールドレコーディングに関心を持つ市民が集まり、まちや自然にまつわる音を記録・共有するクラブです。

> 私たちは、音を通して地域文化や自然環境を見つめ直し、普段は意識されにくい“さいたまの音風景”を記録し、展示やWebアーカイブとして発信していきます。

評価:

1文目と2文目で「記録・共有」「記録し、発信」が重複しています。2文目は「なぜやるのか」「何につなげるのか」に寄せると役割分担が明確になります。

修正文案:

> さいたま市を拠点に、フィールドレコーディングに関心を持つ市民が集まり、まちや自然にまつわる音を記録・共有するクラブです。

> 聴くことを通して、地域文化や自然環境を見つめ直し、展示やWebアーカイブとして残していきます。

### 6. Event本文は硬い表現を少しほぐせる

現文:

> 夏の夕刻、大宮の街から氷川の森へ。自然へと歩を進め、変化する音の風景を身体・五感をつかって体感します。

問題:

「自然へと歩を進め」はやや文語的で、イベント告知としては硬く感じます。「身体・五感」も意味が重なります。

修正文案:

> 夏の夕刻、大宮の街なかから氷川参道、大宮公園周辺へ。歩きながら、場所ごとに変わる音の風景を体感します。

### 7. ナビゲーター紹介の文が長い

現文:

> さいたま市を拠点にフィールドレコーディングを通して、まちの音を記録、共有、発信するプロジェクトを行う団体。

問題:

「を通して」「を行う団体」が重なり、やや説明調です。

修正文案:

> さいたま市を拠点に、まちの音を記録・共有・発信するフィールドレコーディング団体。

現文:

> さいたまのまち音収集家・音楽家。環境音楽のリリースやライブ、DJ、世界の環境音プロジェクトに参加し、サウンドスケープの創作・研究など国際的に活動。

問題:

実績が多く、魅力はありますが、一文に詰め込みすぎです。イベント参加者に必要なのは「どんな人が案内するか」なので、少し整理すると読みやすくなります。

修正文案:

> さいたまのまち音収集家・音楽家。環境音楽の制作、ライブ、DJ、サウンドスケープの創作・研究を行い、国内外の環境音プロジェクトにも参加。

## 重要度低

### 8. Projectの見出しはやや重い

現文:

> さいたまの“音”風景 録音ワークショップ＆成果展示発表会

評価:

「成果展示発表会」は事業名として必要なら残してよいですが、一般読者には少し硬く長い印象です。トップページの見出しでは、少し柔らかくしてもよいです。

修正文案:

> さいたまの音風景を記録し、展示する

または:

> 録音ワークショップから成果展示へ

### 9. Sound Mapの説明は「なぜ聴くのか」が少し足りない

現文:

> 地図上の録音地点を選ぶと、Livingroom TapesのSoundCloud音源と録音メモが切り替わります。

評価:

機能説明としては明快です。一方で、はじめて見る人には「この地図で何を感じてほしいのか」が少し弱いです。

修正文案:

> 地図上の録音地点を選ぶと、Livingroom TapesのSoundCloud音源と録音メモが切り替わります。場所ごとに異なる、さいたまの音の表情を聴き比べられます。

### 10. Map注記の「sample locations」は公開ページでは弱い

現文:

> Map data © OpenStreetMap contributors / sample locations

問題:

サイトが本公開に近い状態なら、「sample locations」は仮感が出ます。

修正文案:

> Map data © OpenStreetMap contributors / locations estimated from SoundCloud descriptions

日本語に寄せる案:

> Map data © OpenStreetMap contributors / 録音地点はSoundCloudの表記から推定

### 11. Listening Notesは良いが、ページ内で役割が説明されていない

評価:

短い詩的な記録として雰囲気は良いです。ただ、前後の文脈なしに出てくるため、イベントの記録なのか、サンプルコピーなのかが少し曖昧です。

見出し修正文案:

> 音風景のメモ

導入を足す案:

> 録音やサウンドウォークの中で見つけた、短い聴取メモです。

### 12. Programの文体が不統一

現文:

> 大宮駅東口、氷川参道、その周辺をフィールドに録音を行う。

> 音源、地図、写真、映像、サウンド・インスタレーションを組み合わせた展示。

問題:

同じProgram内で、「行う」という常体と、体言止めが混在しています。他セクションは丁寧語が多いので、ここも揃えると自然です。

修正文案:

> 大宮駅東口、氷川参道とその周辺をフィールドに録音を行います。

> 音源、地図、写真、映像、サウンド・インスタレーションを組み合わせて展示します。

### 13. Joinのボタン文言は少し整理できる

現文:

> 録音ワークショップ申込み / 準備中

問題:

スラッシュ区切りは管理画面のように見えます。参加者向けには自然な表記にしたいです。

修正文案:

> 録音ワークショップは募集開始前です

現文:

> 展示発表会の情報を見る

問題:

リンク先がProgramのため、展示発表会だけの情報に飛ぶわけではありません。

修正文案:

> 今年度の予定を見る

### 14. Footerの「Sample sound map」は削除推奨

現文:

> Field Recording Club Saitama / Sample sound map

問題:

公開ページとしては仮感が強く、団体サイトの信頼感を下げます。

修正文案:

> Field Recording Club Saitama

または:

> Field Recording Club Saitama / Soundscape Archive

## 表記統一の推奨

以下はサイト内で統一したほうがよい表記です。

| 現在の表記揺れ | 推奨 |
| --- | --- |
| `フィールドレコーディングクラブさいたま` / `Field Recording Club, Saitama` / `Field Recording Club Saitama` | 日本語本文は `フィールドレコーディングクラブさいたま`、英語表記は正式名称を1つに統一 |
| `“音”風景` / `音風景` | 基本は `音風景`。強調が必要な見出しだけ `“音”風景` |
| `氷川の森` / `大宮公園` / `氷川参道` | ルート説明では `氷川参道、大宮公園周辺` など具体地名を併記 |
| `Webアーカイブ` / `Webアーカイブ化` | 見出しは `Webアーカイブ`、動詞は `Webアーカイブとして残す` |
| `Peatixで申し込む` | このままでよい。CTAとして明快 |

## そのまま残したい表現

以下はサイトの個性として有効です。

- `耳をひらくと、さいたまは別の風景になる。`
- `7月18日、聴くために歩く。`
- `まちが呼吸しているようにも感じる。`

ただし、詩的な文はページ全体に多く置きすぎると要点がぼやけるため、ヒーロー、Listening Notes、Joinに絞るのがよいです。イベント説明やProgramでは、具体性を優先するほうが申し込みにつながります。

## 実装優先度

すぐ直したいもの:

1. Programの集合場所から「仮」を外す、または全体を「予定」に統一する
2. ルート表記を「大宮駅周辺から氷川参道、大宮公園周辺へ」に統一する
3. Footerの「Sample sound map」を削除する
4. Event本文に「録音経験や専用機材は不要」を追加する

次に直したいもの:

1. Aboutの2文目を短くする
2. ナビゲーター紹介を簡潔にする
3. Sound Map説明に「聴き比べ」の価値を足す
4. Programの文体を丁寧語に統一する

## まとめ

現在の文言は、活動の雰囲気とイベントの詩的な魅力を伝える力があります。大きく書き換えるよりも、告知に必要な具体情報を少し足し、表記揺れと冗長な箇所を整えるのがよいです。

特に「耳をひらくと、さいたまは別の風景になる。」は強いコピーなので、その周辺の説明文は短く具体的にしたほうが、コピーの余韻が生きます。
